# NttdataPruebaTecnica

Proyecto de prueba tecnica NttData.

Para ejecutar localmente debe descargar dos proyectos:

- NttDataBack: el codigo fuente lo encontrara en el siguiente link de github -> https://github.com/rafaelospinacastro/NttDataBack

- NttDataFront: el codigo fuente lo encontrara en el siguiente link de github -> https://github.com/rafaelospinacastro/NttDataFront

Para el proyecto NttDataBack debe correr sobre el puerto 8090, eso con el fin de que se consuma correctamente el frontend.

Las versiones son las siguientes:
Backend: NET 8
Frontend: Angular 17.3.7
Para que funcione correctamente debe tener estas versiones descargadas en el computador donde se correrÃ¡ el back y en front.

El pipeline se encuentra en el archivo -> pipeline_pruebatecnica.yml, que se encuentra en este proyecto.
